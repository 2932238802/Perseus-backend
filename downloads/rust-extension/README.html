Rust Support for Perseus

This extension provides robust and lightning-fast support for the Rust programming language in the Perseus IDE.

"Empowering developers to build reliable and efficient software, directly from Perseus."

✨ Features

  • One-Click Project Creation: Seamlessly integrate with cargo new to bootstrap projects.
  • Native Build System: Execute cargo build directly from the IDE's command palette.
  • Zero Configuration: Works out of the box with your existing Rust toolchain.

🚀 Quick Start

After installing this extension, you can create a new project using the internal command system:

// Example internal command mapping
{
    "command": "rust.create",
    "args": ["my_awesome_project"]
}

⚙️ Requirements

You must have the official Rust toolchain installed on your system. If you haven't installed it yet, please run:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
